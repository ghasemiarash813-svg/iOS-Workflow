# RestaurantCostApp

SwiftUI اپلیکیشن مدیریت دستور پخت، بانک مواد اولیه و محاسبه خودکار هزینه‌ها برای رستوران‌ها.

## قابلیت‌ها
- مدیریت دستورهای نامحدود با مقیاس‌بندی تعداد نفرات
- بانک جامع مواد اولیه با تاریخچه قیمت و ضریب ضایعات
- محاسبه خودکار هزینه مستقیم، سربار، هزینه متغیر و قیمت پیشنهادی
- نمایش ریال و تومان با قالب‌بندی استاندارد ایران
- داشبورد تحلیلی، ماتریس مهندسی منو و تحلیل سناریو
- پشتیبانی از RTL و دارک مود

## اجرا
- پروژه را در Xcode باز کنید و target اصلی را اجرا کنید.
- داده‌های نمونه برای ۱۰ غذای محبوب ایرانی در اولین اجرا بارگذاری می‌شوند.
